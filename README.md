# MAE116 - Metodos Matematicos em Biologia I

Esse repositório contem as notas e outros documentos para as aulas do curso MAE116 Métodos Matemáticos em Biologia I do curso de Biofísica na UFRJ para o ano 2026.

O curso é desenvolvido para estudantes do primeiro período de Graduação de Biofísica.

## Objetivos
- Conhecer os conceitos básicos da matemática utilizados em áreas científicas
- Traduzir um problema de biologia, física ou química para linguagem matemática
- Compreender um problema e resolvê-lo utilizando os métodos matemáticos adequados
- Passar da linguagem cotidiana para a linguagem matemática e vice-versa.

## Ementa

Números, funções, sequências e limites. Continuidade e deriva. Aplicações de derivadas. Integral
definida, técnicas de integração, aplicações da integral definida.

### Critérios de avaliação
- Para se apresentar à prova final, é imperativo apresentar uma taxa de **presença em aula superior a 75%**.
- Durante o semestre, serão realizadas **3 provas avaliativas** escritas sendo os 2 melhores notas consideras para a média (soma das 2 melhores notas, dividido por 2).
- No final do semestre, teria a **Prova Final**, prova escrita de 2 horas.
- Estudantes com média menor que 5,0 serão reprovados.
- No formato avaliativo proposto, **não há segunda chamada** para as atividades (uma vez que o aluno conta 3 instrumentos avaliativos e ainda poderá excluir a menor nota obtida).
- Terá uma **segunda chamada apenas para Prova Final**, para alunos que apresentarem comprovante de ausência durante a primeira chamada.
- A nota do exame final terá um coeficiente mais elevado do que as atividades de avaliação contínua.
